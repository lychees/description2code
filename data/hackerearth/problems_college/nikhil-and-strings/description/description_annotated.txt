Nikhil is one of the birghtest student in his whole batch and because of this reason every teacher hands over all the work to him. This time his task is to find the middle Character(s) of the string in lower case.

Input

First line inputs Number of test cases 'N'.

Next N lines have N strings.

Output 

Print the middle Character(s) of each test case in different line.

Constraint

N ≤ 20

Size of string ≤ 1000

SAMPLE INPUT
2
Hello
WORLD

SAMPLE OUTPUT
l
r
