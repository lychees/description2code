Ricky gives a number 'n' to John. Now Ricky wants John to generate Z with symbol "*" such that it spans n rows.

Example:

Input : n=3

Output :

***
  *
***

Input : n=4

Output :

****
   *
  *
****

Constraint :

3 ≤ n ≤20

SAMPLE INPUT
5

SAMPLE OUTPUT
*****
   *
  *
 *
*****

Explanation

The 'Z' is printed across 5 lines as shown above.
