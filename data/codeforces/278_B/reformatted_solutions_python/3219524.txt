import sys
from itertools import product
from string import ascii_lowercase as al
n = int(raw_input())
items = [raw_input().strip() for _ in xrange(n)]
lis = list(al)
for y in product(al, al):
    strs = "".join(y)
    lis.append(strs)
for x in lis:
    if all(x not in y for y in items):
        print x
